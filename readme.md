### Git Ignore Template

Keep OS generated files, archives and node modules from cluttering your repos.

```

   |\          .(' *) ' .
   | \        ' .*) ignore .'*
   |(*\      .*(//  . *) .
   |___\       // (. '*
   ((("'\     // '  * .
   ((c'7')   /\)
   ((((^))  /  \
 .-')))(((-'   /
    (((()) __/'
     )))( |
      (()
       ))

```
